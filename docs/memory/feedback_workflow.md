## Не генерировать SVG иконки для карточек
Пользователь попросил SVG-частицы для ethos карточек, результат был "некрасивый". Переключились на фото из AI-генераторов.
**Why:** SVG-иконки выглядели дёшево для премиального бренда.
**How to apply:** Для визуального контента предлагать фото/рендеры, не SVG-иконки.

## AI-генераторы создают нереалистичную технику
Генераторы изображений плохо рисуют реальное GPU-железо (несуществующие модели).
**Why:** Пользователь заметил что сгенерированные GPU не похожи на настоящие.
**How to apply:** Для GPU-изображений использовать low-poly/стилизованный подход, не фотореализм.

## Промпты должны быть разнообразными
Sprocket (дракон-ревьюер) раскритиковал одинаковые промты для 4 карточек.
**Why:** Каждый промт должен иметь уникальный сюжет, ракурс и настроение.
**How to apply:** При генерации серии промтов — делать каждый уникальным по композиции.

## Пользователь любит видеть результат на мобильнике
Часто просит "выпусти в вайфай" через `npx serve`.
**Why:** Проверяет как выглядит на реальном устройстве.
**How to apply:** После значимых визуальных изменений предлагать запустить сервер.
