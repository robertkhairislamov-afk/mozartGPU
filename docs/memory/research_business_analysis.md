## Рынок GPU-аренды 2026
- 2023: $3.34 млрд → 2032: $26-34 млрд (CAGR ~25-30%)
- "Парадокс инфраструктуры ИИ": затраты растут линейно с ростом запросов (не как SaaS)
- Remote Access Security Act (январь 2026) — экспортный контроль на облачные GPU = новый сегмент клиентов

## Конкуренты — слабые места
- **Vast.ai**: нет SLA, Docker-only (нет bare-metal), хост может уйти offline
- **RunPod**: низкая CPU производительность, потеря данных при остановке, запутанный UI хранилища
- **SimplePod**: данные удаляются при завершении аренды, нет статических IP
- **Spheron**: скудный выбор преднастроенных окружений

## Три целевых сегмента
1. **Корпоративные клиенты из санкционных юрисдикций** — самый платежеспособный, платят за анонимность + крипто
2. **Операторы ИИ-агентов** — эластичная инфраструктура, посекундная тарификация, надёжность
3. **Web3/инди разработчики** — преднастроенные среды, фиксированный прайс, крипто-оплата

## Unit Economics (маржа по GPU)
- RTX 4090: закупка $0.29-0.33 → продажа $0.45-0.50 → маржа **35-50%**
- A100: $0.60-1.19 → $1.80-2.00 → **50-100%+**
- H100: $1.54-2.25 → $3.20-3.80 → **60-100%+**
- B200: $2.81-3.75 → $7.00-8.00 → **85-180%** ← центр прибыли!
- LTV:CAC целевой: 2.0x+ (SaaS стандарт 3.0x, но маржа ниже)

## Скрытые издержки арбитража
- "Горячий резерв" — оплаченные прогретые инстансы для быстрого деплоя
- Storage tax — Vast.ai тарифицирует диск при паузе
- SLA-компенсации — могут съесть **до 30% прибыли** без AIOps
- Egress fees — до $7,500/мес для кластера 4×H100

## Двухуровневая стратегия
**Этап 1 — Арбитраж (без CAPEX):** ресейл Vast.ai → валидация PMF → cash flow
**Этап 2 — Своё железо (CAPEX):** B200 в Чонбури → маржа x2-3

**Триггеры перехода:** утилизация >20-30% 24/7 + LTV клиентов >14 мес + платежи поставщикам >$30K/мес

## Таиланд (Чонбури) — для Этапа 2
- Электричество $0.02-0.06/кВт*ч (vs $0.19 в США)
- Free Trade Zones — беспошлинный ввоз серверов
- BOI-льготы до 8 лет при социальных обязательствах
- Риски: водный кризис, требования BOI по обучению тайцев
- Первый кластер 8×B200: $350-400K, окупаемость 14 мес при полной загрузке

## Три архитектурных must-have:
1. Reverse Proxy со статическими эндпоинтами
2. Persistent Storage 25 Гбит/с
3. Библиотека преднастроенных образов + AIOps auto-failover
