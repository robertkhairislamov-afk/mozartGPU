## Ситуация: РФ гражданин, живёт в Тайланде, GPU ресейл бизнес

## Оптимальная стратегия (3 фазы):

### Фаза 1 — Крипто без юрлица (неделя 1-2, ~$400)
- **BTCPay Server** self-hosted — 0% комиссия, USDT/USDC, без KYC
- **Bybit Card** — конвертация крипты в USD для оплаты vast.ai
- DTV виза ($270) — 180 дней без стресса
- Личный тайский банковский счёт (Kasikorn/Krungsri)
- Валидация бизнес-модели до инвестиций в юрлицо

### Фаза 2 — Thai Limited Company (месяц 2-3, ~$3-5K)
- Регистрация через юрфирму (Siam Legal / Juslaws / Integrity Legal)
- Номинальная схема: 51% тайцы + Preference Shares + Shareholders Agreement
- Bangkok Bank бизнес-счёт
- **Omise/Opn Payments** (не Stripe — риск блокировки РФ бенефициара)
- Non-B виза + Work Permit
- Стоимость: ~75-150K бат ($2-4K) + ~$1.5-2K/год операционных

### Фаза 3 — Масштабирование (месяц 6+, при $20K+ MRR)
- **BOI заявка** → 100% иностранное владение + 0% налог на 3-8 лет
- EU entity (Estonia OÜ или UK Ltd) для Stripe и EU/US клиентов
- LTR виза для долгосрочной стабильности

## Почему НЕ Estonia сразу:
- E-Residency для РФ граждан — одобрение ~20-30%, срок 6-12 мес
- Эстонские банки отказывают РФ гражданам в большинстве случаев
- Stripe может заблокировать при РФ бенефициаре

## Санкционные риски:
- Обычный РФ гражданин (не SDN List) — не под прямыми санкциями
- Vast.ai с тайским адресом + нероссийской картой — риск низкий
- Тайские банки лояльнее западных к РФ гражданам
- Omise/2C2P — без OFAC concerns

## Налоги:
- Тайланд: если >183 дней/год = налоговый резидент. Доход облагается 0-35% при ввозе в TH
- РФ: нерезидент (>183 дней за рубежом) — нет обязательства по НДФЛ
- Thai Company: CIT 20% (первые 300K бат = 0%), VAT 7% при обороте >1.8М бат/год

## Визы:
- Сейчас: DTV ($270, 180 дней)
- При Thai Company: Non-B + Work Permit
- Долгосрок: LTR (10 лет) или Smart Visa (4 года)

## Крипто-лицензия: НЕ нужна для приёма оплаты за услуги (нужна только для бирж/ICO)

**Why:** Крипто-старт позволяет запуститься за дни без затрат. Thai Company — через 2-3 мес при подтверждённом спросе.
**How to apply:** Не торопиться с юрлицом — сначала валидация через крипто, потом Thai Company при $5K+ MRR.
